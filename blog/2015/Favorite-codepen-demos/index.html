Favorite Codepen demos

02 Jun 2015 , Taggged in: codepen

If you don’t know what is Codepen - it is an HTML, CSS, and JavaScript code editor in your browser with instant previews of the code you see and write. It is a searchable trove of your own creations, and a world of other people’s creations. Want to show off your latest creation and get feedback from other coders? Do you like to explore the code of other front-end developers? CodePen is great for that.

Thanks to Alex Vazquez (@alexquez), Tim Sabat (@timsabat) and Chris Coyier (@chriscoyier) for creating this, finally you can have your own online playground.
